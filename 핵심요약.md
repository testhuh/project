# 🎯 멜론 → YouTube 재생목록 자동 추가 시스템 - 핵심 요약

## ✨ 실제 구현 방식

### 1️⃣ **Playwright + Chrome 자동화**
```
✅ API 없음 (YouTube API 불필요)
✅ Playwright로 Chrome 브라우저 제어
✅ 실제 UI를 마치 사람처럼 자동 조작
✅ headless=false로 실시간 진행 상황 확인
```

### 2️⃣ **멜론 공유 링크 (로그인 불필요)**
```
✅ 멜론 공개 공유 링크로 직접 접근
✅ GUI에서 URL 입력 (하드코딩 또는 입력창)
✅ 로그인 없이 곡 정보 크롤링 가능
✅ 크롤러가 직접 HTML 파싱
```

### 3️⃣ **.env 환경변수 관리 (YouTube만)**
```env
# ⭐ YouTube 자격증명만 .env 사용
YOUTUBE_EMAIL=your_email@gmail.com
YOUTUBE_PASSWORD=your_password

# Chrome 설정 (선택사항)
CHROME_PATH=/usr/bin/google-chrome
HEADLESS_MODE=false  # 브라우저로 보기 가능

# 실행 설정
DELAY_BETWEEN_ACTIONS=2
MAX_RETRIES=3
TIMEOUT=30
```

### 실행 순서
```bash
# 1. 의존성 설치
pip install -r requirements.txt
playwright install chromium

# 2. .env 파일 생성 및 설정
echo "MELON_PLAYLIST_URL=..." > .env
echo "YOUTUBE_EMAIL=..." >> .env
echo "YOUTUBE_PASSWORD=..." >> .env

# 3. 실행
python main.py
```

### Playwright 핵심 코드
```python
from playwright.sync_api import sync_playwright
from dotenv import load_dotenv
import os

load_dotenv()

# Chrome 시작 (headless=false로 보기)
with sync_playwright() as p:
    browser = p.chromium.launch(
        headless=False,  # 👈 중요: 브라우저 화면 표시
        executable_path=os.getenv('CHROME_PATH')
    )
    page = browser.new_page()
    
    # 멜론 공유 링크로 이동 (로그인 불필요!)
    melon_url = os.getenv('MELON_PLAYLIST_URL')
    page.goto(melon_url)
    
    # 곡 정보 파싱
    songs = page.query_selector_all('tr.lst50')
    # ... 처리
```
