# 🎯 멜론 → YouTube 재생목록 자동 추가 시스템 - 핵심 요약 (최종 수정)

## ✨ 실제 구현 방식

### 1️⃣ **Playwright + Chrome 자동화**
```
✅ API 없음 (YouTube API 불필요)
✅ Playwright로 Chrome 브라우저 제어
✅ 실제 UI를 마치 사람처럼 자동 조작
✅ headless=false로 실시간 진행 상황 확인
```

### 2️⃣ **멜론 공유 링크 (로그인 불필요)**
```
✅ 멜론 공개 공유 링크로 직접 접근
✅ GUI에서 URL 입력 (입력창 또는 하드코딩)
✅ .env에 저장하지 않음
✅ 로그인 없이 곡 정보 크롤링 가능

멜론 URL은 어떻게 전달?
├─ GUI 입력창에서 입력
├─ 코드에 직접 하드코딩
└─ 명령줄 인자로 전달
```

### 3️⃣ **.env 환경변수 (YouTube만 사용)**
```env
# ⭐ YouTube 자격증명만 .env에 저장
YOUTUBE_EMAIL=your_email@gmail.com
YOUTUBE_PASSWORD=your_password

# Chrome 설정 (선택사항)
CHROME_PATH=/usr/bin/google-chrome
HEADLESS_MODE=false  # 브라우저로 보기 가능

# 실행 설정
DELAY_BETWEEN_ACTIONS=2
MAX_RETRIES=3
TIMEOUT=30
```

### Playwright 핵심 코드
```python
from playwright.sync_api import sync_playwright
from dotenv import load_dotenv
import os

load_dotenv()

class MelonCrawler:
    def __init__(self, melon_url):  # 👈 GUI에서 입력받음
        self.melon_url = melon_url
    
    def collect_songs(self):
        with sync_playwright() as p:
            browser = p.chromium.launch(headless=False)
            page = browser.new_page()
            
            # GUI에서 입력받은 멜론 URL로 이동
            page.goto(self.melon_url)  # 👈 .env가 아니라 GUI 입력
            page.wait_for_load_state('networkidle')
            
            # 곡 정보 파싱
            songs = page.query_selector_all('tr.lst50')
            # ... 처리

class YouTubeAutomation:
    def __init__(self):
        # .env에서 YouTube 자격증명만 로드
        self.email = os.getenv('YOUTUBE_EMAIL')
        self.password = os.getenv('YOUTUBE_PASSWORD')
    
    def login(self):
        # YouTube 로그인
        page.fill('input[type="email"]', self.email)
        page.fill('input[type="password"]', self.password)
        # ...
```
